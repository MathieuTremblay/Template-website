Le HTML de A à Z

Le HTML (ou hyper text markup language) est un langage que permet de créer des sites web.

Malheureusement, il ne viens pas seul. En effet, le HTML permet de créer la base des pages web, ce qui veut dire: les textes, images, menu,etc.

Pour complèter le HTML, vous devez lui inclure d'autre language comme le CSS, le javascipt, le php, etc.

Pour commencer, une page doit contenir une base de code comme vous le voyez ci-dessous.

1. 2. 3. 4. 5. 6. 7.
<!DOCTYPE html> <html> <head> </head> <body> </body> </html>

Commencons tout d'abort par décortiquer ce code.

1.
<!DOCTYPE html>

Le DOCTYPE est là pour indiquer qu'elle version de l'html vous utiliserez. Dans ce tutoriel nous utiliserons la version 5 qui s'écrit <!DOCTYPE html>La doctype, içi: html5, indique au navigateur le type de document que vous avez écrit..

2. 3. 4. 5. 6. 7.
<html> <head> </head> <body> </body> </html>

Les lignes lignes 2 et 7 indique respectivement le début et la fin du document html. Les lignes 3 et 4 indique quant'à eux le head de votre page. Ces dans cette section que vous insererez votre titre principal et les liens vers les documents css et javascript. Pour conclure, les lignes 5 et 6 crée la zone ou vous insererez vos codes html afin de créez votre page.

En HTML, vous devez utiliser des balises afin de spécifier ce qu'est chaque objet. Voici les principaux que vous utiliserez dans le début du tutoriel:

Texte

1. 2. 3. 4. 5. 6. 7.
<title> <!-- Indique le titre de votre page, a placer dans la section head --> <a href=""> <!-- Indque un lien que mène vers une autre page ou à une hancre --> <p> <!-- Indique une zone où est écrit du text, un paragraphe --> <h1> <!-- Indique un titre. Allant de h1 à h6, il désique respectivement un titre allant de très important à peu important --> <b> <!-- Mettre un text en gras --> <code> <!-- Vous permet d'afficher des codes comme vous le voyez présentement sur vos pages --> <i> <!-- Mettre un text en italique -->

Les division

1. 2. 3. 4. 5. 6. 7. 8. 9.
<div> <!-- Crée une division où vous pouver inscrire votre contenu --> <span> <!-- Idem --> <header> <!-- Zone en-tête de page --> <footer> <!-- Zone en bas de la page --> <content> <!-- Diviser un contenu --> <article> <!-- diviser votre text en différent article --> <table> <!-- insérer un tableau a colonne --> <nav> <!-- créer votre menu pour naviquer sur votre site --> <aside> <!-- insererez une zone, généralement sur le côté de la page -->

Voici les principales balises HTML que vous utiliserez pour débuter. Passons immédiatement a un exemple de code simple d'une page HTML
